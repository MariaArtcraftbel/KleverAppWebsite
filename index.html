Клевер — Управление видеозаписями

Автоматическое копирование видео с USB-дисков

Что умеет «Клевер»?

  • Автоматически находит подключённые USB-накопители (флешки, внешние диски)
  • Ищет видеофайлы (.mp4, .avi, .mov, .mkv) по заданным расширениям
  • Сортирует и копирует видео по дате создания и устройству
  • Создаёт папки автоматически с учётом даты и имени устройства
  • Поддержка ручной и автоматической очистки дисков (по заполненности)
  • Оповещение при переполнении памяти на накопителе
  • Пробный период 5 дней без ограничений
  • Платная активация через код
  • Интерфейс с прогресс-барами, логами и статусами устройств
  • Встроенный файловый менеджер для просмотра записей
  • Поддержка запуска на Windows (рекомендуется 10/11)
  • Скрытая работа — приложение можно свернуть в трей
  • Блокировка / разблокировка дисков вручную
  • Простая установка

Как работает «Клевер» — пошаговая инструкция

1. Скачайте и запустите приложение

Скачайте архив с программой по кнопке выше, распакуйте и запустите файл .exe. При первом запуске начинается 5-дневный пробный период.

2. Подключите USB-накопитель

Программа автоматически определит подключённый накопитель (флешку, внешний диск). При этом будет показан список доступных устройств и файлов.

3. Добавьте новое устройство

В интерфейсе нажмите «Добавить устройство», введите его имя (например, «Toyota Cam 001»). Это имя будет использоваться для сортировки видео по папкам.

4. Копирование файлов

Программа находит видеофайлы на накопителе, сортирует их по дате и копирует в папку, соответствующую выбранному устройству. Вы увидите ход копирования через прогрессбар.

5. Очистка и блокировка диска

Если диск переполнен, программа предложит удалить старые файлы. Также вы можете вручную заблокировать диск, чтобы избежать случайных действий.

6. Просмотр файлов

Через встроенный файловый менеджер можно просматривать скопированные видео, открывать папки и проверять сохранённые записи.

7. Активация

После окончания пробного периода программа предложит ввести код активации. Инструкция будет показана в отдельном окне при нажатии на кнопку "купить приложение".
